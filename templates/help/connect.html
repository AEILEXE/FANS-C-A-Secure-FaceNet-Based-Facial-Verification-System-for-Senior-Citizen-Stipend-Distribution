Troubleshooting
Domain (fans-barangay.local) not working?
Use the LAN IP address instead. {% if server_lan_url %} Copy {{ server_lan_url }} into the browser. {% endif %} The domain requires a one-time hosts file setup on each client device, but the IP address works immediately.
Camera not working?
Browsers only allow camera access on https:// or localhost. Use {{ server_domain_url }} (HTTPS) from client devices. The plain IP URL (http://) cannot access the camera.
Cannot reach the server at all?
Make sure all devices are on the same Wi-Fi or LAN network. Mobile data (4G/5G) will not work. The server and client must be on the same router.
