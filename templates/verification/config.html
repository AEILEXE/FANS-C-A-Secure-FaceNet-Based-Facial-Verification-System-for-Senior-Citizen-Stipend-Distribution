About the Similarity Threshold

The threshold determines how closely a live face must match the registered embedding to be verified.

  • Higher (e.g. 0.85): More strict — fewer false accepts, more false rejects
  • Lower (e.g. 0.65): More lenient — fewer false rejects, more false accepts
  • Recommended: 0.70 – 0.80 for senior citizens
